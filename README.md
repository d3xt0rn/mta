# mta
simple ascii video player for linux made by AI
<img width="641" height="679" alt="image" src="https://github.com/user-attachments/assets/90e8fe2a-6f03-4497-8bd2-69215c9681a2" />
# dependencies
- ffmpeg
- gcc dev packages

# use
```bash
mta[ver] video.mp4 -flags
#like: mta14 video.mp4 -256 -F60
```
## recommended terminal settings
- font size: 2-4px
- terminal: kitty as fullscreen

# for help list
add flag -h at the end
